### 从 v0.1.0 到 v1.0.0 的完整升级

- **v0.1.0**：仅有 1 个 `pack-onboarding` skill
- **v1.0.0**：16 个 skills 全面上线

### 16 个 Skills 清单

| Skill | 功能 |
|---|---|
| `output-html` | HTML §7B PPTX-Ready 快速输出 |
| `output-pptx` | 真实 PowerPoint 文件输出 |
| `output-pdf` | PDF 格式输出 |
| `output-image` | 图片格式输出 |
| `brand-system` | 品牌适配系统（73+ 品牌库） |
| `marp-slides` | 22 个 Marp 示例模板 |
| `deep-research` | 深度研究内容生成 |
| `designer` | 设计师视角优化 |
| `data-viz` | 数据可视化 |
| `narrative` | 叙事结构设计 |
| `review` | 审阅与改进 |
| `rehearsal` | 演讲演练 |
| `citation` | 引用管理 |
| `websearch` | 联网搜索 |
| `ppt-beautify` | 美化润色 |
| `onboarding` | 上手引导 |

### 三阶段设计流

1. Template matching（模板匹配）
2. Design DNA（设计基因提取）
3. 8-dim Design Critic（8 维设计评审）
