# Chat 输入框快捷动作栏上线（Quick start / Clear Context / Compress / Tips）

## 核心宣传点

对话框上方新增 4 个快捷按钮，一键清除上下文、压缩对话、获取使用提示，让 AI 对话更顺畅。
