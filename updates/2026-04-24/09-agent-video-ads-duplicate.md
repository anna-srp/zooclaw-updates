# 新 Agent 上架：Video Ads Duplicate — 一句话复刻爆款广告视频

## 核心宣传点
无需拍摄、无需剪辑，提供一条参考广告 + 产品信息，AI 自动生成全新产品广告视频。支持自动抓取 Amazon/任意电商产品信息、搜索爆款参考视频、两种生成路径（风格复刻/智能商品替换），并能生成超过 15 秒的长视频。

## 原始内容

feat(video-ads-duplicate): new agent pack for ad video duplication (#101)

### 核心公式
`参考广告视频 + 产品信息（图片 / 标题 / 描述）= 你的新广告`

### 主要功能
1. **自动获取产品信息**（Amazon & 任意电商页面，三级回退）
2. **爆款广告素材库搜索**（Facebook 系、TikTok、独立站）
3. **两条生成路径**：
   - Path A · 风格复刻：AI 分析分镜 → 重写脚本 → 生成首帧 → Seedance 2.0 续生完整视频（产品外观 100% 还原，规避版权风险）
   - Path B · 智能商品替换：参考视频 + 产品图一次性交给 Seedance，近 1:1 保留运动/构图
4. **专业级广告文案改写**：3 秒 Hook + 真实使用体验 + 核心卖点 + CTA
5. **长视频 Chunk 方案**（> 15 秒，通过尾帧→首帧串联保证镜头连贯）
6. 支持 9:16 竖版（TikTok/Reels/Shorts），人脸首帧自动走 portrait 回退
