# ZooClaw 更新日志


## 2026-04-23

### 🚀 新功能

- **支持私有自定义 Agent 目录**
  团队现在可以管理自己专属的私有 Agent 列表，不对外公开。

- **iOS 语音输入全面升级**
  录音时显示实时波形动画，支持取消/确认，工具栏交互更流畅。

- **iOS 聊天界面全面重构**
  切换为自研的 ChatListView，滚动更流畅，渲染更准确，输入框更稳定。

### 🛠 产品更新

- **重新订阅后 Bot 自动升级到最新版本**
  重新订阅后不再需要手动升级，开笱即用最新能力。

### 🐛 Bug 修复

- **修复 Twitter Connector 连接卡死问题**
  Twitter-v2 Nango proxy 偶发连接卡死问题已修复，Twitter 相关功能更稳定。

- **PPTX 预览文字间距修复**
  含自定义字符间距的幻灯片现在显示正常。

## 2026-04-22

- 🚀 **Specialist Manager v1.0 上线 + 项目惯例路由器**
  全新 Specialist Manager Skill 上线：让你的 AI 团队有专属协调员，自动调度专家、维持项目惯例，多 Agent 协作更丝滑。

- ✨ **PPTX Skill 升级：支持形状级精细编辑**
  PPTX Skill 新增形状级 CLI，可对幻灯片中任意形状进行精细增量操作，PPT 编辑效率大幅提升。

- 🎬 **Chameleon Skill：默认关闭水印 + 支持 1080p 参考视频**
  Chameleon 视频生成默认不再添加水印，同时支持 1080p 高清参考视频输入，生成质量更上一层楼。

- 🐛 **PPTX 预览修复：OOXML 表格现在正常渲染**
  修复含真实表格数据的 PPTX 幻灯片预览空白问题，现在所有 OOXML 表格都能正确显示。

- 🎉 **Seedance 2.0 上新弹窗正式上线**
  Seedance 2.0 视频生成 Agent 上线弹窗，点击即可跳转 Vibe Drama agent 详情页体验最新视频生成能力。

- 🔗 **新功能：未雇佣 Agent 深链智能引导雇佣流程**
  访问未雇佣 Agent 的深链不再静默回退，而是智能引导到 Agent Studio 雇佣流程，降低用户探索 Agent 的门槛。

- 💬 **聊天工具调用展示升级：更丰富的文字、emoji、折叠和计时器**
  聊天界面中 Agent 调用工具的展示全面升级：动作描述更丰富、支持折叠、显示耗时计时器，让你清楚知道 Agent 在做什么。

- 🐛 **修复：侧边栏底部导航不再被 Agent 列表遮挡**
  修复 Agent 列表过长时底部导航（AI Specialists Hub 等）被推出视野的问题，侧边栏操作更流畅。

- ⚡ **性能优化：聊天背景图 880KB→31KB，首屏更快**
  聊天页背景图从 880KB JPEG 压缩为 31KB WebP，首屏加载速度显著提升（LCP 优化）。
