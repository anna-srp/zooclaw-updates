# ZooClaw Changelog

> 面向用户的更新日志，按时间倒序排列。

---

## 2026-04-21

### 🤖 Agent 更新

**vibe-drama Agent 升级至 v1.0.9**
vibe-drama 短剧创作 Agent 升级到 v1.0.9，工作流程持续优化，短剧生成体验更流畅。

### 🚀 新功能

**用 IQ 进度条直观展示 AI 能力状态**
当积分不足使用基础模型时，界面不再显示难懂的文字提示，而是以渐变色 IQ 进度条直观表示 AI 当前的能力水位。

**Mattermost 附件支持直接预览（无需下载）**
在 Mattermost 消息中收到 PDF、PPTX、Word、Excel 等文件附件时，点击即可在侧边栏内联预览，无需下载。PPTX 渲染质量同步大幅提升。

### 🐛 Bug 修复

**连接状态显示更准确，修复新用户启动报错**
连接状态现在需要 WebSocket 和 Mattermost 同时在线才显示为“已连接”，防止误报；聊天界面不再夾杂内部心跳检测消息；新用户首次使用时不再出现钉包报错。

## 2026-04-19

### 🐛 Bug 修复
