# ZooClaw 更新日志

> 记录每次产品更新，最新的在最上面。

---

## 2026-04-21

### 🤖 Agent 更新

- **vibe-drama 短剧 Agent 升级至 v1.0.9**
  短剧创作 Agent 持续打磨，生成流程更流畅稳定。

### 🚀 新功能

- **AI 能力状态可视化：IQ 进度条**
  积分不足时，界面不再显示一行让人困惑的文字提示，而是用一条渐变色进度条直观告诉你"AI 现在有多聪明"。

- **Mattermost 附件一键预览**
  在 Mattermost 里收到 PDF、PPT、Word、Excel 文件时，点击就能直接在侧边栏预览，不用下载再打开。PPT 预览效果同步大幅提升。

### 🐛 修复

- **连接状态更准确 + 新用户首次使用更顺畅**
  修复了"明明没连上却显示已连接"的状态误报问题；聊天界面不再混入内部系统消息；新用户第一次打开时不再遇到报错页面。

### 🐛 修复

## 2026-04-18

### 🚀 新功能

- **批量积分发放 + 补偿弹窗**
  运营侧可以批量向用户发放积分，发放时用户会收到弹窗通知，第一时间知道自己获得了奖励。

- **iOS 回复更流畅，连接更稳定**
  iOS 客户端现在支持 bot 流式回复（像打字机一样逐字出现），支持 emoji 反应，掉线后重新连接也更快、更稳定。

### 🐛 修复

- **修复一个导致大量错误日志堆积的问题**
  修复了 Mattermost 连接异常时触发大量系统报错的 bug，整体运行更稳定。
