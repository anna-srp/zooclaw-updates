# ZooClaw 更新日志

> 记录每次产品更新，最新的在最上面。

---

## 2026-04-21

### 🤖 Agent 上架/更新

- **vibe-drama 短剧 Agent 升级至 v1.0.9**
  短剧创作流程更完善：风格锁定更稳定，生成前可预览每一幕的提示词，有昂贵步骤时会先请你确认，避免意外消耗。

### 🚀 新功能

- **AI 能力状态可视化：IQ 进度条**
  积分不足时，不再显示让人困惑的文字提示，而是用一条进度条直观展示当前 AI 能力状态。

- **Mattermost 附件一键预览**
  在 Mattermost 里收到 PDF、PPT、Word、Excel 文件时，点击即可直接预览，无需下载。

### 🐛 Bug 修复

- **连接状态更准确 + 新用户首次使用更顺畅**
  修复了"明明没连上却显示已连接"的状态误报；新用户第一次打开时不再遇到报错页面。

## 2026-04-16

### 🚀 新功能

- **版本升级提示 + Changelog 独立页面**
  有新版本时会弹出升级提示，新增独立 Changelog 页面，随时查看完整历史更新。

  可以对消息加表情了。工具步骤支持折叠，流式回复动画更顺滑。

- **Seedance 2.0 上新弹窗**
  首次进入聊天页会看到 Seedance 2.0 上新提示，一键雇佣 Vibe Drama Agent 开始创作。每位用户只出现一次。

### 🐛 Bug 修复

- **Agent 更新后不再自动重置会话**
  Agent 重新部署后不再自动清空当前对话，更新后需要手动开新对话才会切换到新版本。
