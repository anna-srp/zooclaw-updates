# ZooClaw Changelog

## 2026-07-02

### 🚀 新功能

- **应用市场支持直接购买付费 Agent 包**：现在可以在 Agent 管理页直接购买付费 Agent 包——点击 Purchase 走 Stripe 安全支付，付款成功后自动完成安装，已购买的包会正确显示为可安装/卸载状态。

### 🛠 Skill 上架/更新

- **新增「知识库检索」Skill**：Agent 现在可以直接检索你所在组织上传的内部知识库文档（含图文段落），问内部资料不用再翻文件，答案直接来自你自己的知识库。

### 🐛 Bug 修复

- **修复 Windows 桌面版构建兼容问题**：ZooClaw Windows 桌面版恢复正常构建与运行，修复了跨平台命令兼容与平台识别问题，Windows 用户可以正常安装使用桌面端。
- **修复 Firefox 下偶发「无法验证会话」黑屏**：网络瞬断或浏览器请求偶发失败时会自动退避重试登录校验，不再直接黑屏要求手动重试。
- **修复部分老账号缺失组织信息无法正常使用**：系统现在会自动修复缺失组织信息的早期账号，登录后即可正常使用。

### ⚡ 体验优化

- **知识库支持删除文档，并在设置中新增入口**：知识库页面可直接删除自己上传的文档（仅所有者可删），设置导航新增「Knowledge Base」入口，状态文案也更易懂。
- **官网首页焕新**：落地页整体改版为清爽浅色主题，首屏与导航栏重新设计，页面加载更轻快、观感更现代。
- **企业版登录页焕新并更名为 ZooWork**：business.zooclaw.ai 登录与验证码页面全面视觉升级，品牌统一为 ZooWork | BUSINESS。
- **首页侧边栏玻璃质感 UI 打磨**：侧边栏玻璃表面、导航选中态、Logo 与个人资料卡视觉升级，暗色下不再出现刺眼白块。
- **BossClaw 全新 App 图标**：BossClaw 活动页图标与 favicon 统一换成新的藏青底金爪品牌标识。
- **Agent Builder 在 Claw 未运行时给出明确提示**：不再一直显示「工作区准备中」，而是清晰说明原因并给出可操作提示。
