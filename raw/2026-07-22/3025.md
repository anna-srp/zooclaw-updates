## Summary
- keep the shared `data-e2e-overlay` contract as the primary popup selector
- fall back to the four selectors that existed before #3023 when an environment has not deployed the contract yet
- add an isolated Playwright project covering the shared contract, legacy production paywall, and required-action safety

## Root cause
Production E2E runs branch test code against the currently deployed application. The helper from #3023 only recognized the new shared selector, but production had not deployed those application attributes yet. The subscription paywall therefore remained open and intercepted chat composer clicks until the long test timeout expired.
